# CV Website - Lukas Kania

Professional CV website built with vanilla HTML, CSS, and JavaScript.

## Quick Start

### Direct Browser Access (No installation needed)

Simply open `index.html` in your web browser by double-clicking the file.

## Export to PDF

1. Open the website in your browser
2. Press `Ctrl+P` (Windows/Linux) or `Cmd+P` (Mac)
3. Select "Save as PDF" as the destination

## Structure

```
lukekania.github.io/
├── index.html      # Main HTML structure
├── styles.css      # All styling and print styles
├── script.js       # JavaScript for interactivity
├── photo.jpg       # Profile photo
└── README.md       # This file
```

## Browser Compatibility

Works in all modern browsers:

- Chrome/Edge (recommended for PDF export)

- Safari
